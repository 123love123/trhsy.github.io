# 网站使用next主题

上一篇我记录了怎么使用hexo+next搭建GitHub的个人网站，飞机票

# 准备

首先我们要下载安装next 如何下载这里给两个方法，https://github.com/iissnan/hexo-theme-next这个github地址进去之后直接下载 另外一种方法我个人比较喜欢，为啥因为快，因为这样才是程序员下载东西的方法(▼へ▼メ) 首先我们先在d盘新建一个文件夹名字叫next,你随便都行等要用到 win+r输入cmd打开dos命令行切换到刚才你建的文件夹下

D:
cd D:/next
git clone https://github.com/iissnan/hexo-theme-next themes/next
1
2
3

下载完了打开看到 嘻嘻，其实next主题安装很简单直接把你下载的next整个文件夹拷贝至站点目录的 themes 目录下，之后修改配置文件就可以了你想要啥样该啥样 之后我们打开_config.yml搜索theme,修改为

theme: next
1

我加入的功能 百度分享 阅读统计我用了LeanCloud和不蒜子 评论 Valine 话不多说我先上我的next主题下的_config.yml配置
