TortoiseGit配合git使用

# TortoiseGit配合git使用

Git安装详情请移步 (opens new window) TortoiseGit下载地址 (opens new window) 下载完了两个文件这里安装的是64位的不知道32位有没有区别，应该没有

# 安装 TortoiseGit

安装界面化管理工具“TortoiseGit”。 双击打开“TortoiseGit-2.7.0.0-64bit.msi”。全部默认选择，一路下一步完成安装。 完了安装汉化包 双击打开“TortoiseGit-LanguagePack-2.7.0.0-64bit-zh_CN.msi”。全部默认选择，一路下一步完成安装。 安装完成之后win+d快捷键到桌面右键TortoiseGit-设置 修改语言 在设置页面中，点击“常规设置”然后单击“重新运行首次启动向导”。 设置语言下一步 下一步 设置git.exe路径

设置git仓库用户名邮箱，这里用的是github的 生成密钥 选择rsa点击generate生成，鼠标需要在key的格子里移动加速生成 把生成的可以复制添加到github仓库中 登录到github右上角点击头像settings 左边菜单ssh and GPG keys 新建new ssh key 把生成的key容复制进去 保存路径推荐C:\Users\asus.sshc盘用户下.ssh中
